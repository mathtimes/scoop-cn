## 添加应用库

添加本应用库。

```powershell
scoop bucket add scoop-cn https://ghproxy.com/github.com/mathtimes/scoop-cn
```

## 安装应用

```powershell
scoop install scoop-cn/APPNAME
```

## 查看帮助

要了解 Scoop 的更多用法，请查看 [Scoop 官网](https://scoop.sh/)

```powershell
scoop help
```
